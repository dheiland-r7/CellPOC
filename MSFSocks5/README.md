This is a metasploit framework module that support communicating network traffic/tools via a socks5 proxy thru serial to Quectel cellular module using AT commands. This module allows user to route traffice into the IoT devices trusted ecosystem

It is recommend that the module be run on a stand alone metasploit client and not be used via metasploit on a VM. Often load sharing of USB/Serial with VM can have communication related problem leeding to poor data throughput. degrading the systems or causing application hangs and timeouts.
